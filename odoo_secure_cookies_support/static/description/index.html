Secure Cookies Support

Adds the Secure flag to every session and CSRF cookie served by Odoo, so cookies are only transmitted over HTTPS, helping you meet modern browser and regulatory security requirements.

Usage

  1. Make sure your Odoo instance is reachable via https://.
  2. Go to Settings / Technical / System Parameters.
  3. Create or update the parameter secure_cookies and set its value to 1 to enable secure cookies.
  4. (Optional) Change the value to 0 to disable the feature.

Tip: Users may need to clear their browser cookies after the change so the Secure attribute is applied to session cookies.
